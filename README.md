# Чат-приложение (Frontend)

![Test and Deploy](https://github.com/Madmaxim22/chat-frontend/workflows/Test%20and%20Deploy/badge.svg)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue?logo=github)](https://madmaxim22.github.io/chat-frontend/)

Это клиентская часть веб-приложения чата, реализованная с использованием HTML, CSS и JavaScript. Приложение обеспечивает интерактивное общение пользователей в реальном времени с помощью WebSocket-соединений.

## Функциональные возможности

- Регистрация пользователя через модальное окно с вводом никнейма
- Отображение списка активных пользователей в боковой панели
- Обмен сообщениями в реальном времени через WebSocket
- Отображение сообщений с указанием отправителя, времени и даты
- Возможность покинуть чат с автоматическим уведомлением других участников
- Адаптивный интерфейс с визуальным выделением собственных сообщений

## Архитектура

Проект состоит из следующих основных компонентов:

- `ChatManager.js` - основной класс управления логикой чата
- `ViewChat.js` - класс для отображения и управления интерфейсом чата
- `HttpLoader.js` - класс для выполнения HTTP-запросов к серверу (регистрация пользователя)
- `WebSocketLoader.js` - класс для работы с WebSocket-соединением
- `index.js` - точка входа, инициализирующая все компоненты

## Требования к системе

- Node.js
- Веб-браузер, поддерживающий WebSocket и современные стандарты JavaScript

## Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите режим разработки:
   ```bash
   npm start
   ```

3. Для сборки продакшн-версии:
   ```bash
   npm run build:prod
   ```

## Использование

1. При открытии приложения появится модальное окно для ввода никнейма
2. После успешной регистрации (никнейм должен быть уникальным) откроется интерфейс чата
3. В левой части отображается список активных пользователей
4. В правой части - окно чата, где можно отправлять и получать сообщения
5. Для выхода из чата используйте кнопку "Выйти из чата"

- Регистрация пользователя: `POST https://chat-backend-cntc.onrender.com/new-user`
